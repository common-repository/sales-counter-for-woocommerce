== Description ==
WooCommerce Sales Counter Plugin For WordPress.



== Installation ==

1. Install as regular WordPress plugin.
2. Go your Plugins setting via WordPress Dashboard and activate it.
3. After activating the plugin, go to Dashboard-> Products and you see two Sales and Earnings.
